User

HOST: https://api.magmakick.io

게임 사용자 관리에 사용한다.

Resource Group 

사용자 등록 

POST
/user/add/{appFingerPrint}

게임 사용자 등록에 사용한다.

  • UUID:asdfer… (string, required) - 기기 고유 식별자

  • NickName:magmafriend (string, required) - 사용자 별칭

  • Locale:ko-kr (string, required)

    사용자 언어 및 국가, [국가]-[언어]형식을 사용하기를 권장한다.

  • Parameters
  • appFingerPrint
    string (required) 

    app 고유 식별자
